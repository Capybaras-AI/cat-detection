Keywords: Computer Vision

### Datasets

- [CIFAR-10](https://www.cs.toronto.edu/~kriz/cifar.html):
    - Number of records: 60000
    - Image size: 32x32x3
    - Number of classes: 10
    - Train test split: 80/20
    - Authors: [1]
